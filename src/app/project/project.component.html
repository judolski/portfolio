Computer Based Test(CBT)
The images below show interface samples of a project that I developed. It is an examination web app that requires users to register, login and take exams. it also consists of an admin page where admin can create questions, access results and modify information of users. The project was developed using Angular as frontend, NodeJS and MongoDb as Backend.
